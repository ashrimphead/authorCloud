Description
--------------
OG subgroups module is integrated with Drupal's book module. Book module allows a user to create an outline of nodes.
When the outline is consisted of group nodes, members and group posts can propagate along the outline (refered as subgroups outline).
Private groups and their children groups are kept hidden from users without the correct access permissions.

Dependencies
--------------
* Book module - core module.
* Organic groups - http://drupal.org/project/og
